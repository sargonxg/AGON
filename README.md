## the aco ontology

eight primitives, locked. nineteen edge types. no rdf. no triples. just rust.

```
ACTOR        individuals В· organizations В· states В· coalitions
CLAIM        asserted fact / evaluation / normative statement
INTEREST     underlying goal motivating positions (fisher/ury)
CONSTRAINT   rule / norm / structural limit
LEVERAGE     resource / dependency / capability
COMMITMENT   promised future action (subject + deadline + status)
EVENT        dated or orderable occurrence
NARRATIVE    coherent framing across claims, per actor
                + PatternFinding (darvo, gaslighting, 4 horsemen)
                + AffectMarker, RepairAttempt, BidForConnection
```

## why rust

```
  в”Ңв”Җ schema-constrained extraction
  в”Ӯ     vertex ai is the typed sensor. JSON in, JSON out. rust validates.
  в”Ӯ
  в”ңв”Җ canonicalization (alias graph, dedup)
  в”Ӯ     petgraph + blake3 + hnsw_rs. no Python startup tax. zero-copy.
  в”Ӯ
  в”ңв”Җ deductive inference (ascent / datalog)
  в”Ӯ     parallel closure on 50k primitives in < 1s. one binary.
  в”Ӯ
  в”ңв”Җ contradiction detection (z3)
  в”Ӯ     real SMT solver. unsat cores trace back to source spans.
  в”Ӯ
  в”ңв”Җ negotiation modelling (good_lp В· lp + milp)
  в”Ӯ     batna / zopa as feasibility problems. solver in-process.
  в”Ӯ
  в”ңв”Җ pattern recognition (rule-based + neural, both)
  в”Ӯ     fastembed for embeddings. candle / burn for in-binary neural inference.
  в”Ӯ
  в””в”Җ sovereign reasoning
        no remote inference service besides extraction.
        deterministic. reproducible. auditable.
```
